What is a responsive site and why is responsiveness important?


A responsive site is website designed to adapt to fit the specific screen capabilities it is drawn up on. For instance, the same site -- if responsive -- can have altered designs for desktops and smart phones, while based off the same code instructions.



What is mobile first design, and why is it important?


Mobile-first design is an instruction and industry standard for designing responsive sites. By starting with the smallest screen, like the ones on mobile smart phones, the progression to fill larger pages like tablets, netbooks, and desk tops can progress more smoothly and beautifully.



What are frameworks? What are their pros and cons?


Frameworks are tools that offer a developer a template to work off of. These typically include code for responsiveness and a few other key features, including grid structure, typography, buttons, and more.



What is a wireframe and why do we use it?


A wireframe is the first "mock up" that a developer/designer team does to sketch out their basic plan for the website. Wireframes are a crucial first step in getting designer, developer, and client on the same page as the project is approached.
