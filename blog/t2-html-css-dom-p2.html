What is a grid-based design?


Grid based design is a design method that breaks up a page into a percentage of it, and allows for organized navigation of thought using text, images, and blank space to guide the reader.



What's all the hype about responsive webpages?


Responsive pages are an absolute must for today's digital world. From desktop to tablets to iPhones, each screen size requires different adaptabilities from different components. Responsive sites are ones that can flex from 1000px to 30px and still look great doing it.



What is semantic structure?


Semantic structure is the way that tags are broken up in HTML, including Headers, images, and forms.
