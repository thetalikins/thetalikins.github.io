How does Javascript compare with HTML and CSS?


Where CSS and HTML are static, load-on-command web page features, Javascript is a resposive language that reacts and actively re/animates the page while it's open. It's adaptable in its command features; where HTML/CSS say "do this," JS says "If this, than do that. If not, do this instead."
